JavaScript functions practice

Defining and running functions

Reusable procedures that allow us to write reusable, modular code

Defining a function:

function funcName() {
    // do something
}

Running/executing a function:

funcName()

All methods are functions, more on that later.

Arguments

We can also write functions that accept inputs, called arguments

Adding inputs to a function:

function funcName(input, input2, etc) {
    // do something
}

When we create the function, the inputs we require are called parameters, but when we execute/run the function are then called arguments.

Example

// Defining the function
function greet(firstName) {
    console.log(`Hey there, ${firstName}`);
}

// Running the function
greet("Carlos"); // Hey there, Carlos
